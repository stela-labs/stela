# Security Policy

## Supported Versions

Stela is pre-1.0. Security fixes target the latest published minor version.

## Reporting a Vulnerability

Please report suspected security issues privately by opening a GitHub security advisory on the repository.

Do not file public issues for vulnerabilities involving:

- database access control bypasses
- unsafe SQL execution
- credential exposure
- workflow state corruption
- unintended replay of side effects

Include a minimal reproduction, affected version or commit, and expected impact when possible.
